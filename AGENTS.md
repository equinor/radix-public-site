# Agents.md

Radix-public-site is a documentation site for end users of the open source platform Radix built by Equinor.

## Policy

Avoid writing about Equinor specific settings and configuration

When editing `.md` or `.mdx` files, always apply the radix-docs skill to review changes for style, tone, and content compliance.

## Skills

| Skill              | Path                                        | Description                                                                                            |
| ------------------ | ------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| maintain-agents-md | `.agent/skills/maintain-agents-md/SKILL.md` | Sync AGENTS.md when skills are added, removed, or modified.                                            |
